# opnsense_firewall_alias (Data Source)

Look up an existing firewall alias on OPNsense by its UUID. Use this data source to reference alias attributes in other resource configurations without importing the alias into Terraform management.

## Example Usage

```terraform
# Look up an existing firewall alias by UUID
data "opnsense_firewall_alias" "k8s_services" {
  id = "12345678-1234-1234-1234-123456789012"
}

# Reference the alias attributes in other resources
output "alias_name" {
  value = data.opnsense_firewall_alias.k8s_services.name
}

output "alias_content" {
  value = data.opnsense_firewall_alias.k8s_services.content
}
```
