# opnsense_kea_dhcpv6_subnet (Data Source)

Look up an existing Kea DHCPv6 subnet on OPNsense by UUID without importing it into Terraform management.

## Example Usage

```terraform
# Look up an existing Kea DHCPv6 subnet by UUID
data "opnsense_kea_dhcpv6_subnet" "example" {
  id = "12345678-1234-1234-1234-123456789012"
}

output "kea_dhcpv6_subnet_id" {
  value = data.opnsense_kea_dhcpv6_subnet.example.id
}
```
