# opnsense_ddclient_account (Data Source)

Look up an existing dynamic DNS account on OPNsense by UUID without importing it into Terraform management.

The `password` field is intentionally omitted because it is sensitive/write-only and is not populated from OPNsense read responses.

## Example Usage

```terraform
# Look up an existing dynamic DNS account by UUID
data "opnsense_ddclient_account" "example" {
  id = "12345678-1234-1234-1234-123456789012"
}

output "ddclient_account_id" {
  value = data.opnsense_ddclient_account.example.id
}
```
