# opnsense_haproxy_server (Data Source)

Look up an existing HAProxy server on OPNsense. Use this data source to read attributes by UUID without importing the object into Terraform management.

## Example Usage

```terraform
# Look up an existing haproxy server by UUID
data "opnsense_haproxy_server" "example" {
  id = "12345678-1234-1234-1234-123456789012"
}

output "haproxy_server_id" {
  value = data.opnsense_haproxy_server.example.id
}
```
