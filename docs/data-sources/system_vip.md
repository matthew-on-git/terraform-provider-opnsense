# opnsense_system_vip (Data Source)

Look up an existing virtual IP on OPNsense. Use this data source to read attributes by UUID without importing the object into Terraform management.

The CARP `password` resource attribute is intentionally unavailable in this data source because OPNsense does not reliably return it on read.

## Example Usage

```terraform
# Look up an existing system vip by UUID
data "opnsense_system_vip" "example" {
  id = "12345678-1234-1234-1234-123456789012"
}

output "system_vip_id" {
  value = data.opnsense_system_vip.example.id
}
```
