# opnsense_ipsec_child (Data Source)

Look up an existing IPsec child on OPNsense. Use this data source to read attributes by UUID without importing the object into Terraform management.

## Example Usage

```terraform
# Look up an existing ipsec child by UUID
data "opnsense_ipsec_child" "example" {
  id = "12345678-1234-1234-1234-123456789012"
}

output "ipsec_child_id" {
  value = data.opnsense_ipsec_child.example.id
}
```
