# opnsense_firewall_category (Data Source)

Look up an existing firewall category on OPNsense. Use this data source to read attributes by UUID without importing the object into Terraform management.

## Example Usage

```terraform
# Look up an existing firewall category by UUID
data "opnsense_firewall_category" "example" {
  id = "12345678-1234-1234-1234-123456789012"
}

output "firewall_category_id" {
  value = data.opnsense_firewall_category.example.id
}
```
