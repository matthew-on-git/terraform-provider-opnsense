# opnsense_firewall_category (Resource)

Manages a firewall category on OPNsense. Categories organize firewall rules into logical groups for easier management and visibility.

## Example Usage

```terraform
# Manage a firewall category for web traffic rules
resource "opnsense_firewall_category" "web_traffic" {
  name  = "web_traffic"
  color = "0000ff"
}

# Manage a category for management access rules
resource "opnsense_firewall_category" "management" {
  name  = "management"
  color = "ff0000"
}

# Manage a category with no color
resource "opnsense_firewall_category" "monitoring" {
  name = "monitoring"
}
```

### Required

- `name` (String) Name of the category. Must be unique. Commas are not allowed.

### Optional

- `color` (String) Color as 6-digit hex (e.g., `ff0000` for red). Empty for no color.
