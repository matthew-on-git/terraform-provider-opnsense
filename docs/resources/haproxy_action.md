# opnsense_haproxy_action (Resource)

Manages an HAProxy action on OPNsense. Actions connect ACL matches to routing behavior such as selecting a backend, denying requests, redirecting HTTP to HTTPS, or setting request headers. Link action UUIDs from `opnsense_haproxy_frontend.linked_actions`. Requires the `os-haproxy` plugin.

## Example Usage

```terraform
# Route matching host-header traffic to a backend pool.
resource "opnsense_haproxy_action" "route_api" {
  name        = "route-api"
  type        = "use_backend"
  use_backend = opnsense_haproxy_backend.api.id
  linked_acls = [opnsense_haproxy_acl.api_domain.id]
  test_type   = "if"
}

# Redirect cleartext HTTP requests to HTTPS.
resource "opnsense_haproxy_action" "redirect_https" {
  name     = "redirect-to-https"
  type     = "http-request_redirect"
  redirect = "scheme https code 301"
}

# Set a header before forwarding to the backend.
resource "opnsense_haproxy_action" "forwarded_proto" {
  name               = "set-x-forwarded-proto"
  type               = "http-request_set-header"
  set_header_name    = "X-Forwarded-Proto"
  set_header_content = "https"
}
```

## Schema

### Required

- `name` (String) Name of the HAProxy action.
- `type` (String) Action type. Supported values: `use_backend`, `map_use_backend`, `http-request_deny`, `http-request_redirect`, `http-request_set-header`.

### Optional

- `deny_status` (Number) HTTP status code for `type = "http-request_deny"`.
- `description` (String) Description of the HAProxy action.
- `http_request_option` (String) Raw HAProxy option text for HTTP request actions when OPNsense requires an action-specific option string.
- `linked_acls` (Set of String) Set of HAProxy ACL UUIDs used as conditions for this action.
- `map_use_backend_default` (String) Default backend UUID for `type = "map_use_backend"` when no map entry matches.
- `mapfile` (String) Map file UUID for `type = "map_use_backend"`.
- `operator` (String) How multiple linked ACLs are combined: `and` or `or`. Defaults to `and`.
- `redirect` (String) Redirect rule text for `type = "http-request_redirect"`, for example `scheme https code 301`.
- `set_header_content` (String) Header value or format string for `type = "http-request_set-header"`.
- `set_header_name` (String) Header name for `type = "http-request_set-header"`.
- `test_type` (String) ACL match polarity: `if` or `unless`. Defaults to `if`.
- `use_backend` (String) Backend UUID for `type = "use_backend"`.

### Read-Only

- `id` (String) UUID of the HAProxy action in OPNsense.
