# opnsense_haproxy_acl (Resource)

Manages an HAProxy ACL (Access Control List) rule on OPNsense. ACLs define match conditions used for routing decisions — matching on host headers, URL paths, SNI, source IPs, and more. ACLs are referenced by HAProxy actions to control traffic routing. Requires the `os-haproxy` plugin.

## Example Usage

```terraform
# Match requests where Host header starts with a domain
resource "opnsense_haproxy_acl" "api_domain" {
  name       = "api-domain"
  expression = "hdr_beg"
  hdr_beg    = "api.example.com"
}

# Match requests to a specific path prefix
resource "opnsense_haproxy_acl" "admin_path" {
  name       = "admin-path"
  expression = "path_beg"
  path_beg   = "/admin"
}

# Match by SNI for SSL passthrough
resource "opnsense_haproxy_acl" "ssl_domain" {
  name       = "ssl-domain"
  expression = "ssl_fc_sni"
  ssl_fc_sni = "secure.example.com"
}

# Match by source IP
resource "opnsense_haproxy_acl" "trusted_src" {
  name       = "trusted-source"
  expression = "src"
  src        = "10.0.0.0/8"
}
```

## Schema

### Required

- `expression` (String) Match expression type (e.g., `hdr_beg`, `hdr`, `path_beg`, `path`, `ssl_fc_sni`, `ssl_sni`, `src`, `nbsrv`, `custom_acl`).
- `name` (String) Name of the ACL rule.

### Optional

- `custom_acl` (String) Raw HAProxy condition (pass-through).
- `description` (String) Description of the ACL.
- `hdr` (String) HTTP Host header exact match.
- `hdr_beg` (String) HTTP Host header starts with this value.
- `hdr_end` (String) HTTP Host header ends with this value.
- `nbsrv_backend` (String) UUID of the backend to check for minimum usable servers.
- `negate` (Boolean) Invert the match condition. Defaults to `false`.
- `path` (String) URL path exact match.
- `path_beg` (String) URL path starts with this value.
- `src` (String) Source IP address match.
- `ssl_fc_sni` (String) SNI TLS extension matches (locally deciphered).
- `ssl_sni` (String) SNI TLS extension matches (TCP inspection).

### Read-Only

- `id` (String) UUID of the HAProxy ACL in OPNsense.

## Import

Import is supported using the ACL UUID from OPNsense.
