# opnsense_firewall_nat_outbound (Resource)

Manages an outbound NAT (source NAT) rule on OPNsense. Outbound NAT rules control how internal traffic is translated when leaving through an interface, typically translating private source addresses to the WAN IP.

## Example Usage

```terraform
# NAT internal network to WAN IP
resource "opnsense_firewall_nat_outbound" "lan_to_wan" {
  interface   = "wan"
  source_net  = "10.0.0.0/24"
  target      = "wanip"
  description = "LAN outbound NAT"
}

# NAT with static port preservation (for SIP/VoIP)
resource "opnsense_firewall_nat_outbound" "voip" {
  interface       = "wan"
  protocol        = "udp"
  source_net      = "10.0.10.0/24"
  target          = "wanip"
  static_nat_port = true
  description     = "VoIP outbound with static port"
}

# No-NAT rule (exclude specific traffic from NAT)
resource "opnsense_firewall_nat_outbound" "vpn_no_nat" {
  interface       = "wan"
  source_net      = "10.0.0.0/24"
  destination_net = "10.1.0.0/24"
  target          = "wanip"
  no_nat          = true
  description     = "Skip NAT for VPN traffic"
}
```

## Schema

### Required

- `interface` (String) Outbound network interface (e.g., `wan`).
- `target` (String) Translation target IP or alias (e.g., `wanip`, interface IP, or specific address).

### Optional

- `categories` (Set of String) Set of category UUIDs assigned to this rule.
- `description` (String) Description of the rule.
- `destination_net` (String) Destination network (`any`, CIDR, or alias).
- `destination_not` (Boolean) Invert destination match. Defaults to `false`.
- `destination_port` (String) Destination port or range. Empty for any.
- `enabled` (Boolean) Whether this rule is enabled. Defaults to `true`.
- `ip_protocol` (String) IP version: `inet` (IPv4) or `inet6` (IPv6).
- `log` (Boolean) Log matching packets. Defaults to `false`.
- `no_nat` (Boolean) Suppress NAT for matching traffic. Defaults to `false`.
- `protocol` (String) Protocol (e.g., `any`, `tcp`, `udp`, `TCP/UDP`).
- `sequence` (Number) Rule sequence number (1-999999). Controls evaluation order.
- `source_net` (String) Source network (`any`, CIDR, or alias).
- `source_not` (Boolean) Invert source match. Defaults to `false`.
- `source_port` (String) Source port or range. Empty for any.
- `static_nat_port` (Boolean) Preserve the original source port. Defaults to `false`.
- `target_port` (String) Translated source port. Empty to preserve original.

### Read-Only

- `id` (String) UUID of the outbound NAT rule in OPNsense.
